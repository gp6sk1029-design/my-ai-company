# ライフプランくん SKILL.md

## 自己改善ループ（CLAUDE.mdに準拠）
このエージェントはCLAUDE.mdの方針に従い、タスク完了のたびに振り返りレポートを出力し、SKILL.mdとMEMORY.mdを更新し続ける。ROI評価を毎回行い、費用対効果を最大化する。

---

## プロジェクト目的
家族の生涯キャッシュフローと資産推移を「年齢×口座種別×ライフイベント×投資複利」で可視化し、老後資金の枯渇リスクを数値で把握できる自家製PWAを提供する。

## 対象ユーザー
- 本サービスのオーナー自身（家族構成・副業収入・投資口座を含む）
- 将来：配偶者との共有、子供の教育進路判断、FIRE可否検討

## 生産技術思考の適用

| 原則 | 本ツールでの実装 |
|---|---|
| ムダ排除 | 手計算・Excelのセル依存を排除、入力→即時再計算 |
| 数値判断 | 「なんとなく不安」を「○歳で資産枯渇」に翻訳 |
| 再現性 | 純粋関数 `calc.js` に計算を集約、副作用なし |
| 常改善 | 毎回の利用で入力精度を上げ、前提シナリオを見直す |
| 費用対効果 | 新NISA／特定口座の税引き後リターンを両方出し、意思決定材料に |

## ドメイン知識（忘れずに反映）

### 新NISA（2024〜）
- 年間拠出上限：つみたて枠120万＋成長枠240万＝360万円
- 生涯非課税枠：1,800万円（うち成長枠は1,200万円まで）
- 運用益は非課税、売却益も非課税
- 本ツールでは超過分は特定口座にオーバーフローさせる

### 特定口座（源泉徴収あり）
- 譲渡益20.315%（所得税15% + 住民税5% + 復興特別所得税0.315%）
- 配当も同税率。損益通算可

### 暗号資産
- 雑所得で累進課税（最大55%）
- 本ツールでは簡易化のため20.315%で計算（注釈必須）

### 教育費（本ツールのデフォルト値の根拠）
- 文科省「子供の学習費調査」令和3年度
- 日本政策金融公庫「教育費負担の実態調査」令和5年度
- 大学は4年総額→年額換算、医歯系は6年制総額

### 4%ルール（Trinity Study）
- 退職時資産の4%を毎年取崩すと30年後の枯渇確率が極めて低い
- 本ツールの老後取崩デフォルト

## UI設計の原則

- **モバイルファースト**：すべての入力はプロンプト式でも動くが、将来フォーム化
- **1画面1目的**：ホームはCF＋グラフ、それ以外は入力専用
- **赤字年は視覚化**：単年赤字は薄いピンク、資産枯渇年は濃いピンク
- **スマホから随時更新可能**：入力後すぐホーム画面に反映

## 進化のトリガー（CLAUDE.md踏襲）

| トリガー | 対応 |
|---|---|
| 同じ計算ミスが2回起きた | `calc.js` にテストケース追加、禁止事項に追記 |
| 税制改正 | `config.js` の定数を更新、根拠コメント必須 |
| MF家計簿CSVのフォーマット変更 | `parseKakeiboCsv` の列名検出regex（計算対象/日付/金額/保有金融機関/大項目/振替）を見直し、サンプルをreferenceに保存 |
| MF資産画面のUI変更でOCR精度低下 | `extractBalancesFromOcr`/`classifyAssetLine` のキーワード辞書を拡充、失敗時は一括棚卸しモードに誘導 |
| 新しい資産種別の追加要望 | `assets.kind` を拡張、labelマップも更新 |

## 禁止事項

- ハードコードされた税率を呼び出し側で二重に適用する（課税は calc.js 内のみで行う）
- 複利計算で単年ごとの積立タイミング（月初/月末）を曖昧にする → 年末一括（年12回分をまとめて年末投入）で固定
- インフレを支出のみ・収入ゼロで適用する（収入にも `growthRate` を設定してもらう）
- 根拠のない利回り表示（プリセット以外のデフォルト値を出さない）
- **外部サービスの機能を事実確認せずに実装しないこと**（MF資産CSVのように「出ない」ものを「出る前提」で組むと公開後にUIごと空振りする。ドキュメント or 実データで必ず検証）
- OCR結果を確認なしに反映しない（必ず差分プレビューを挟み、抽出失敗時は手動フォールバックを用意）

## 成功パターン

- 計算を純粋関数 `calc.js` に分離 → UIとロジックを独立改善できる
- IndexedDB + `updatedAt/deletedAt` の論理削除 → 将来のD1同期に備えた
- CSVインポート時に「差分プレビュー」を挟む → 意図しない上書きを防げる
