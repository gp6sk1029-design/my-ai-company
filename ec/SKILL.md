## 2. 出品フロー

### 不用品の場合
```
1. Google Drive「メルカリ」フォルダから商品写真を取得
2. 写真から商品を特定（ブランド・型番・状態を推定）
3. メルカリ相場調査（SOLD品の価格収集）
4. 写真から商品サイズ推定 → 梱包材選定 → 送料計算
5. GO/NO-GO判定（送料込みの利益計算）
6. タイトル・説明文生成
7. 初期価格決定
8. 【承認ゲート】ユーザーに出品内容を提示 → 承認
9. メルカリに出品
10. 在庫DB登録
```

### せどりの場合
```
1. ユーザーが仕入れ候補を提供
2. メルカリ相場調査
3. 利益率計算・GO/NO-GO判定
4. 【承認ゲート】仕入れ判断結果をユーザーに提示
5. GOの場合 → 出品フローへ
```

## 5. 価格設定ルール

### 初期価格
```
推奨価格 = 相場中央値 × 0.93
最低価格 = (仕入れ価格 + 送料 + 最低利益) ÷ (1 - 手数料率)
推奨価格 = max(戦略価格, 最低価格)
```

### 商品タイプ別
| タイプ | 戦略 | 理由 |
|---|---|---|
| 不用品 | 相場の85〜90% | 在庫コスト0なので早く売る |
| せどり | 相場の90〜95% | 利益率30%を確保 |
| オリジナル | 相場の95〜100% | 差別化できるため高め |

### 値下げスケジュール
```
出品日 → 3日後: -5%
       → 7日後: -10%
       → 14日後: -15%
       → 21日後: 再出品
```
- ¥100未満の値下げは¥100に切り上げ
- 最低利益ラインを下回る値下げは自動停止
- 20%以上の値下げは人間承認が必要

## 6. GO/NO-GO判断基準

### 不用品
```
利益 > ¥0 → GO
```

### せどり
```
利益 ≥ ¥500 かつ 利益率 ≥ 30% → GO
```

### オリジナル商品
```
利益 ≥ ¥1,000 かつ 利益率 ≥ 50% → GO
```

### 計算式
```
販売価格 = 相場中央値 × 0.93
手数料 = 販売価格 × 10%（メルカリ）
送料 = 発送方法別の固定値
利益 = 販売価格 - 手数料 - 送料 - 仕入れ価格
利益率 = 利益 ÷ 仕入れ価格 × 100
```

### リスク評価
| リスク | 高リスク | 中リスク | 低リスク |
|---|---|---|---|
| SOLD品数（30日） | 5件未満 | 5〜20件 | 20件以上 |
| 出品中の競合数 | 30件以上 | 10〜30件 | 10件未満 |
| 価格変動幅 | 50%以上 | 20〜50% | 20%未満 |
| 推定販売日数 | 14日以上 | 7〜14日 | 7日未満 |

高リスクが2つ以上 → GO判断でも「注意」フラグ

## 8. 顧客対応テンプレート

### 値下げ交渉
```
【応じる場合】利益基準を満たす
コメントありがとうございます！
¥{新価格}でしたら変更可能です。ご検討いただけますでしょうか？

【断る場合】利益基準を下回る
コメントありがとうございます！
申し訳ありませんが、こちらの商品は現在の価格が限界となっております。

【中間提案】
コメントありがとうございます！
ご希望の価格は難しいのですが、¥{中間価格}でしたらいかがでしょうか？
```

- 10%以内の値下げ → 自動対応可能
- 10〜20%の値下げ → 中間提案
- 20%以上の値下げ → ユーザーに確認

### 購入後メッセージ
```
【購入直後】
この度はご購入ありがとうございます！{X日以内}に発送させていただきます。

【発送完了】
発送が完了しました。追跡番号：{番号}
到着までもうしばらくお待ちください。
```

### エスカレーション必須（人間対応）
- 商品の破損・不良品の申告
- 商品が届かないとの連絡
- 返品・返金要求
- 攻撃的・脅迫的なメッセージ

## 9. 承認ゲート（人間承認が必要なポイント）

| ゲート | 条件 | 内容 |
|---|---|---|
| 出品承認 | 全出品 | タイトル・説明文・価格・写真の最終確認 |
| 高額出品 | ¥10,000以上 | 価格と出品内容の追加確認 |
| 仕入れ承認 | ¥5,000以上 | 利益率・GO/NO-GO判断の確認 |
| 大幅値下げ | 20%以上 | 値下げ理由と新価格の確認 |
| クレーム対応 | 全件 | 対応方針の確認 |

## 11. 禁止事項

- 虚偽の状態表記（傷を隠す表現）
- 過度な煽り文句の乱用
- 他プラットフォームへの誘導
- 禁止出品物の出品支援
- 著作権のある画像の無断使用
- 事実と異なる商品説明
- 個人情報の開示
- メルカリ外での取引誘導
