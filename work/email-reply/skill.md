# メール返信スキル

## 概要
受信メールの情報を受け取り、ビジネスメールとして適切な返信文を日本語で生成する。
返信文は必ず確認用として出力し、自動送信は行わない。

---

## 入力フォーマット

以下の情報を提供してください：

```
【件名】（例：お見積りのご依頼について）
【送信者】（例：株式会社〇〇 田中様）
【本文】
（メール本文をそのまま貼り付け）

【過去のやり取り】（任意）
（関連する過去メールがあれば貼り付け）

【返信の意図】（任意）
（例：承諾する／断る／確認する／お礼を伝える）
```

---

## 処理手順

1. **文脈の読み取り**
   - 件名・送信者・本文から要件を把握する
   - 過去のやり取りがあれば流れを確認する
   - 送信者との関係性（初対面／継続取引など）を推定する

2. **返信方針の確認**
   - 返信の意図が未指定の場合、適切な対応を提案してユーザーに確認する
   - 断る・遅延するなど慎重な対応が必要な場合は一言添えて確認する

3. **返信文の生成**
   - ビジネスメールとして適切な文体・敬語を使用する
   - 以下の構成で作成する：
     - 宛先の書き出し（例：〇〇株式会社 田中様）
     - 冒頭の挨拶
     - 要件への回答・対応
     - 締めの挨拶
     - 署名欄（`【署名】` プレースホルダーで出力）

4. **確認用出力**
   - 返信文を出力した後、以下を明示する：
     - 「この文面でよろしければ、コピーしてご使用ください」
     - 修正したい点があれば伝えるよう案内する
   - **自動送信は絶対に行わない**

---

## 文体ルール

| 状況 | 文体の方向性 |
|------|------------|
| 初めての相手 | 丁寧・堅め |
| 継続取引の相手 | 丁寧・やや柔らか |
| 社内向け | 敬語を保ちつつ簡潔 |
| お断り・遅延 | 丁寧・配慮を込める |
| お礼・承諾 | 明るく・簡潔 |

---

## 出力例

```
---【返信文（確認用）】---

〇〇株式会社
田中 〇〇 様

いつもお世話になっております。
（自分の会社名）の（自分の名前）です。

このたびはお見積りのご依頼をいただき、誠にありがとうございます。
ご要望の内容を確認いたしました。

（本文：要件への回答）

引き続きどうぞよろしくお願いいたします。

【署名】

---

この文面でよろしければ、コピーしてご使用ください。
修正したい箇所があればお知らせください。
```

---

## 注意事項

- 返信文の自動送信は行わない（必ず人間が確認・送信する）
- 金額・納期・契約に関わる内容は特に慎重に扱い、ユーザーに確認を促す
- 個人情報（メールアドレス・電話番号など）はそのまま出力せず伏せる
- 不明な点は返信文を生成する前にユーザーへ確認する
